# 🌱 PDA Paprika

> 자녀의 미래를 함께 준비하는 스마트 투자 관리 서비스, **아이든든**을 만들고 있어요.

<br/>

## 💡 About

**아이든든**은 부모가 자녀의 미래 자금을 체계적으로 준비할 수 있도록 돕는 서비스입니다.  
아동수당 연결, 목표 자금 설정, ETF 투자, 증여 관리 기능을 하나의 흐름으로 제공합니다.

<br/>

## 🌱 Goal

> 부모가 막연하게 느끼는 자녀 미래 자금 준비의 불안감을 줄이고,  
> 목표 설정부터 ETF 투자와 증여 관리까지 이어지는 금융 경험을 제공합니다.
